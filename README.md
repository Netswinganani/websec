A modern, dark-themed security services website built for **Sentricore Security**, showcasing armed response, guarding, and rapid response operations in the West Rand.

---

## 🚨 Overview

This project presents Sentricore Security as a premium, disciplined, and technology-driven protection company with strong visual identity and fast client access.

---

## 🛡️ Key Features

- Hero section with strong security branding
- Services: Guarding, Tech, Response, Access Control, Accountability
- Fleet showcase for operational readiness
- About section with mission-driven values
- 24/7 emergency contact system

---

## 🚗 Fleet Section

Displays operational vehicles and readiness assets:
- Rapid response units
- Security deployment visuals
- Tactical support equipment

---

## 🎨 Design System

- Dark tactical UI
- Electric blue accents
- Glow / scanline animations
- Responsive grid layouts
- Security-themed visual styling

---

## ⚙️ Tech Stack

- HTML5
- CSS3 (custom variables + animations)
- Vanilla JavaScript
